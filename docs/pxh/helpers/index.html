build_remote_pxh_command
Build a shell command that finds and executes pxh on the remote host. When a single explicit path is configured, uses it directly. Otherwise, probes a prioritized list of candidate locations.
default_remote_db_expr
Return a shell expression that resolves the pxh database path on a remote host. Checks XDG path first, falls back to legacy ~/.pxh.
determine_is_pxhs
Determine if the executable is being invoked as pxhs (shorthand for pxh show)
get_relative_path_from_home
Gets the relative path from home directory if the current executable is within it. Returns None if the executable is not in the home directory. Takes optional overrides for testing.
parse_ssh_command
Parse an SSH command string into command and arguments, handling quotes and spaces. Similar to how rsync and other tools parse the -e option.
remote_pxh_candidates 🔒
Build a list of candidate paths to search for pxh on the remote host. The first candidate that exists and is executable will be used.
